## 🔌 PMOD J2 핀 배치 (LED 출력)

### 12핀 PMOD 커넥터 핀아웃

```
        ┌─────────────────────┐
        │    PMOD J2 (Top)    │
        ├─────────────────────┤
  Pin 6 │ VCC (3.3V)          │ Pin 5  GND
  Pin 4 │ C11 → LED[3]        │ Pin 3  D10 → LED[2]
  Pin 2 │ E10 → LED[1]        │ Pin 1  H12 → LED[0]
        ├─────────────────────┤
 Pin 12 │ VCC (3.3V)          │ Pin 11 GND
 Pin 10 │ B11 → LED[7]        │ Pin 9  D11 → LED[6]
  Pin 8 │ E12 → LED[5]        │ Pin 7  B10 → LED[4]
        └─────────────────────┘
```

### FPGA 핀 매핑

| LED 신호 | FPGA 핀 | PMOD 핀 | I/O Standard |
|----------|---------|---------|--------------|
| LED[0] | H12 | J2.1 | LVCMOS33 |
| LED[1] | E10 | J2.2 | LVCMOS33 |
| LED[2] | D10 | J2.3 | LVCMOS33 |
| LED[3] | C11 | J2.4 | LVCMOS33 |
| LED[4] | B10 | J2.7 | LVCMOS33 |
| LED[5] | E12 | J2.8 | LVCMOS33 |
| LED[6] | D11 | J2.9 | LVCMOS33 |

## 📐 LED 모듈 연결

### 옵션 1: Digilent PMOD LED 사용

가장 간단한 방법입니다.

**연결 순서:**
1. KV260 전원 OFF 상태 확인
2. PMOD LED 모듈을 J2 커넥터에 삽입
3. 방향 확인 (VCC/GND 핀 위치)
4. 전원 ON

### 옵션 2: 직접 LED 연결

**필요 부품:**
- LED (5mm 또는 3mm) × 8개
- 저항 (330Ω ~ 1kΩ) × 8개
- 브레드보드
- 점퍼 와이어

**회로 구성:**
```
FPGA Pin ──── 330Ω ──── LED(+) ──── GND
```
